# 面板界面与操作

本页面详细介绍 Fluxor 网页控制面板中，各个功能界面的具体按钮操作与使用细节。

---

## 1. 概览 (Overview)
提供底层内核健康状况与网络流量波动的实时监控。

* **速率与吞吐**：实时展示当前瞬时的上传/下载网速（KB/s 或 MB/s），以及系统本次运行以来的累计上传和下载流量。
* **物理指标卡片**：展示内核的当前物理内存占用、活动连接数以及内核版本。
* **自绘趋势图表**：使用高性能自绘 Canvas 折线图，实时保留并绘制最近 60 秒的流量波动曲线。绿线表示下载，红线表示上传，鼠标悬停可查看具体的点位历史数值。
* **IP 信息与地理位置**：分别显示您的本机和通过内核富强后的出口 IPv4/IPv6 地址及地理位置归属（国家、地区、ISP）。支持一键复制与密文隐藏切换。
* **自定义延迟测试**：支持对预设网站或您自定义的 URL 进行富强延迟测试。

---

## 2. 富强管理 (Proxies)
负责节点选择、测速排序与正则过滤。

* **折叠与展开 (手风琴效果)**：各分流策略组（如 节点选择、谷歌服务、苹果服务 等）默认呈折叠状态。点击策略组卡片可展开下属的所有可用节点。
* **一键测速**：
  * **单节点测速**：点击节点卡片右侧的闪电图标，单独测试该节点的物理延迟。
  * **组测速**：点击策略组卡片右上角的闪电图标，对该策略组内的所有节点并行测速。
  * **全部测速**：点击页面顶部的「测试全部延迟」按钮进行批量测速（限制并发数为 10，防止连接队列拥堵崩溃）。
* **延迟颜色标注**：绿色（150ms 以下）、黄色（300ms 以下）、红色（300ms 以上或超时）。
* **智能节点排序与过滤**：点击策略页面右上角的齿轮按钮：
  * 支持默认排序、按名称、按延迟（低延迟节点自动升序置顶）或按质量评分自动对节点卡片进行排序。
  * 支持输入正则表达式（例如 `香港|新加坡`）过滤不符合条件的节点名称。

---

## 3. 订阅配置 (Subscription)
进行参数管理、规则分流集设置，以及将配置渲染生成并写入内核。

* **全局网络端口**：配置混合富强端口（默认 7890）、控制面板端口（默认 9090）以及透明富强端口。
* **安全密钥配置**：配置外部控制器连接的安全密钥，带显隐切换。
* **规则集选择**：在 Base (标准基础分流) 和 Full (详细增强分流) 中进行选择。
* **订阅模式切换**：选择“融合模式（Merge）”或“切换模式（Switch）”。
* **保存并应用**：修改后必须点击底部的「保存并应用」按钮，系统才会生成全新的 `config.yaml` 配置文件并发送热重载命令，让内核无感切换。

---

## 4. 规则管理 (Rules)
展示当前生效的所有路由匹配规则。

* **检索过滤**：支持在顶部搜索框输入目标域名，检索其命中的分流规则与出站策略。
* **规则临时开关**：每条规则右侧都有一个开关。您可以通过开关随时停用或启用某条特定规则，无需修改底层模板。
* **提供商更新**：支持对 Rule Providers 进行单个更新或一键全部更新（并发上限限制为 3 个并发 Workers）。

---

## 5. 活动连接 (Connections)
监控所有正在通过富强的网络连接连接细节。

* **活跃连接与已关闭连接**：
  * **活跃连接**：实时动态计算出每条连接的瞬时网速，支持排序与检索。
  * **已关闭连接**：保留最近 100 条已断开的连接历史。
* **连接断开与切换修复**：切换新节点后，如果您想让浏览器连接立即走新节点，可以点击顶部的「断开所有连接」按钮，强制内核断开所有当前连接并重新握手。

---

## 6. 日志查看 (Logs)
展示底层的 mi鸿蒙 运行日志。

* **过滤与复制**：可按 debug、info、warning、error 过滤展示特定等级日志，并支持“暂停/恢复滚动”，方便查阅特定日志细节。

---

## 7. 内核配置 (Config)
控制底层的内核系统运行。

* **运行状态控制**：提供一键启动、停止、重启内核的按钮。
* **网卡与系统参数**：控制允许局域网连接、启用 IPv6、修改富强模式、修改日志详细度以及绑定特定的物理网卡。
* **高级清理**：分别提供一键清空 DNS 缓存和一键清空 FakeIP 缓存按钮，用以解决因 DNS 解析污染导致部分网页打不开的问题。
* **内置 DNS 工具**：提供直观的 DNS 解析测试工具。输入域名和类型（A / AAAA），点击即可查看内核解析出的具体 IP，并在查完点击空白处时自动收起手机虚拟键盘。
