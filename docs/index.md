## 项目简介

Fluxor 是一个轻量级的代理面板，旨在为 Mihomo 内核提供便捷的配置管理和运行控制。它通过 Unix Socket 与内核通信，支持：

*   订阅管理（支持融合与切换两种工作模式）。
*   节点测速与连接质量评分系统。
*   规则集（Rule Providers）的精细管理与控制。
*   实时连接状态追踪、按列排序以及手动断开。
*   内核运行日志的流式终端化输出。
*   系统配置管理（端口设置、TUN 虚拟网卡配置、TProxy 透明代理等）。
*   中英双语切换，以及浅色、深色、紫色、粉色等多种主题风格。

## 核心功能特性

### 代理与订阅服务
*   **订阅转换与管理**：支持 HTTP 订阅链接，自动更新节点列表。可选择“融合模式”（所有订阅节点合并）或“切换模式”（每次只激活一个被点击高亮的订阅）。
*   **规则分流管理**：标准模板直接集成并生成内核所需的配置文件。可以在规则界面随时临时禁用或启用单条分流规则，或一键更新所有的 Rule Providers（后台限制 3 个并发 Workers 以免拥堵）。
*   **节点测速与智能排序**：支持策略组独立测速与全局一键批量测速（限制并发数为 10），测速结果按低延迟自动升序置顶。

### 监控与高级控制
*   **概览仪表盘**：展示实时上下行速率、总流量吞吐、内存及连接数，并辅以最近 60 秒的自绘流量走势图。
*   **IP 信息展示**：显示本机的 IPv4/IPv6 以及代理后的出口 IPv4/IPv6 及运营商、地理位置归属地。
*   **网络连接追踪**：实时展示所有正在进行的连接信息，每秒动态计算瞬时上下行速率，支持单条连接阻断或全局阻断重握手。
*   **高级清理与升级**：提供进程的启动/重启/停止控制，提供一键清空 DNS 缓存、清空 FakeIP 缓存以修复域名解析污染，并支持 Geo 数据库在线更新以及内核版本升级。
