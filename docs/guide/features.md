# 功能特性

Fluxor 提供了一系列实用的富强面板管理和订阅功能，主要包括以下几个核心模块：

---

## 核心功能

### 1. 订阅转换与管理
* **节点自动拉取**：输入机场的 Cl~a~sh / mi鸿蒙 订阅 URL，面板可自动拉取节点列表并在后台保存。
* **双模式切换**：支持“融合模式”（将多个订阅的节点打散并混合使用）与“切换模式”（物理隔离，每次只激活一个订阅）。
* **自动定时更新**：支持针对每个订阅设定不同的更新时长。

### 2. 策略组与排序
* **低延迟自动排序**：点击测试延迟后，支持将策略组内的节点按延迟低自动置顶。
* **节点质量自动排序**：选择质量排序后，代理组将支持节点质量评分，评分算法参考自`MetaCubeXD`，点击评分后，支持将策略组内的节点按质量高分自动置顶。
* **测速并发限制**：全局测速和组测速时，后台自动调度并发为最多 10 个 Workers，防止流量拥堵和 Unix Socket 负载过高崩溃。
* **正则过滤**：支持自定义正则表达式，过滤不想要的节点名称|（默认过滤`(套餐|过期|流量|剩余|订阅)`）。

### 3. TProxy 透明富强网关模式
* **高灵活性**：支持劫持局域网设备、本机、本机Docker桥接网络，灵活性远超Tun透明富强模式。
* **绕过列表**：支持设置目的地址例外和源地址例外，让特定的物理主机或目标域名直连，不经过富强。
* **本机富强**：支持通过 nftables 将面板所在主机自身发出的出站流量也进行富强。

### 4. 实时活动连接管理
* **瞬时速率计算**：以快照差分法，每秒精准计算出每条活动连接的瞬时网速。
* **连接阻断**：支持断开指定连接，或一键阻断全局连接以强制更新的节点生效。
* **已关闭连接记录**：保留最近 100 条已断开的连接历史。

---

## 辅助功能与维护
* **内核日志查看**：流式终端展示运行日志，支持日志级别过滤、暂停自动滚动和实时复制。
* **IP 位置探测**：分别获取本机和富强环境下的 IPv4/IPv6，展现其地理位置和运营商归属。
* **DNS 缓存清理**：支持一键清理内核的 DNS 缓存和 FakeIP 映射表，解决常见的解析污染导致国内网站打不开的问题。
* **网络工具**：内置 DNS 查询测试器，可以直接在面板测试域名经由内核解析的结果。
