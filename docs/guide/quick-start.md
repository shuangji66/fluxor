# 快速开始

本指南将帮助您快速上手，使用 Fluxor 进行网络富强配置。

---

## 前提条件

在使用 Fluxor 之前，请确认您的系统满足以下要求：
* **系统环境**：推荐在 Linux（如 Ubuntu/Debian，或飞牛 OS 等 NAS 系统）环境下运行，并设置相应的环境变量。
* **物理依赖**：如果需要开启透明富强，系统必须支持 nftables。
* **内核文件**：已放置对应系统架构的 mi鸿蒙 二进制执行文件，且具备执行权限。

---

## 首次配置三步上手

在网页上打开 Fluxor 面板后，请直接按照以下三个步骤完成您的首次富强激活：

### 第一步：导入订阅并保存应用
1. 在侧边栏导航中，进入「订阅配置」页面。
2. 找到页面中间的「订阅列表」，点击「添加订阅」。
3. 输入信息：
   * **名称**：为您机场的节点起一个英文或拼音别名，支持emoji。
   * **订阅链接**：粘贴您的 cl~a~sh/mi鸿蒙 订阅 URL 链接。
4. 在上方参数配置区：
   * 设定混合富强端口（默认 7890）和面板控制端口（默认 9090）。
   * 选择您需要的分流规则集：
     * **标准 (Base)**：标准分流，匹配速度快，适合轻度用户。
     * **详细 (Full)**：精细分流，分流策略极为丰富，包含流媒体、游戏及广告过滤。
   * 选择订阅模式：**融合模式**（多订阅混合使用）或**切换模式**（点击选中单个订阅）。
5. 点击页面右上方的「保存并应用」按钮。面板会自动合并模板，并启动底层的内核服务。

### 第二步：选择节点
1. 进入「富强管理」页面。
2. 此时会展现系统生成的各个分流策略组。点击页面顶部的「测试全部延迟」，或者点击策略组卡片右上角的小闪电进行并发测速（系统强制并发最大为 10）。
3. 测速完成后，根据右侧红黄绿的延迟数值，点击选定您满意的节点，即可完成切换。
4. 根据喜好可选择`默认`、`按延迟`、`按质量`排序，`按质量`会开启节点质量评分。

### 第三步：开启全局流量接管 (TUN/TProxy)
如果需要您的软路由或服务器系统自动富强所有设备的进出站流量，请前往「内核配置」页面：
* **TUN 模式**：打开 TUN 模式开关，由虚拟网卡接管系统层流量。协议栈可以选择 `System`、`gVisor`、`Mixed`。
* **TProxy 模式**：确认 TProxy 监听端口不为 0 后，打开透明富强开关，并配置您需要的例外 IP 列表。
* **防冲突提示**：TUN 模式与 TProxy 透明富强处于**互斥状态**。开启 TUN 会自动重置并关闭 TProxy；同样，开启 TProxy 会自动清空并关闭 TUN 虚拟网卡。
* **防失联提示**：开启透明代理前需确保本机可靠访问。
