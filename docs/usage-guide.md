# 使用指南

本指南将详细介绍 Fluxor 网页管理面板中，各个功能页面的核心操作与实用使用技巧。

---

## 1. 概览页 (Overview)
提供底层的 Mihomo 内核健康状况以及网络流量吞吐的全局实时监控。

*   **实时速率卡片**：展示当前的瞬时上传网速、瞬时下载网速，以及系统运行以来的累计上传流量和累计下载总量。
*   **物理指标展示**：实时获取当前 Mihomo 内核的物理内存占用、活动连接数量以及内核的精确版本号。
*   **自绘趋势图表**：使用高性能自绘 Canvas 折线图，实时保留并刷新绘制最近 60 秒的流量波动曲线。绿线表示下载速度，红线表示上传速度。
*   **IP 信息展示**：分别显示您的本机 IPv4/IPv6 以及通过内核代理后的出口 IPv4/IPv6 地址与地理归属地（国家、地区、ISP），支持一键复制与密文隐藏切换。
*   **外置面板一键跳转**：如果您需要使用更高级、更全能的外部控制面板，可以在此页直接点击“跳转到 MetaCubeXD”或“跳转到 Zashboard”。系统会自动携带控制端口与 Secret 一键免密跳转。

---

## 2. 代理页 (Proxies)
管理您的策略组、选择节点并进行网络测速。

*   **折叠与展开 (手风琴效果)**：策略组默认折叠以保持页面整洁，点击策略组卡片可展开其下属的所有代理节点。
*   **节点一键切换**：点击任意策略组展开后的节点卡片，即可即时完成切换（带乐观更新，极速反馈）。
*   **代理运行模式**：不仅可以在配置页切换模式，您还可以在代理页的顶部工具栏，一键直接在 Rule（规则分流）、Global（全局代理）和 Direct（直连）三种核心模式间快速切换。
*   **三级并发测速**：
   *   **单节点测速**：点击节点卡片右侧的小闪电图标，单独测试该节点延迟。
   *   **代理组测速**：点击策略组右上角的小闪电图标，对该策略组内的所有节点并行测速。
   *   **全局测速**：点击页面顶部的“测试全部延迟”按钮，对全站所有节点批量测速（并发受限为 10）。
*   **延迟着色机制**：
   *   **绿色 (延迟小于或等于150ms)**：延迟极低，推荐选择。
   *   **黄色 (延迟小于或等于300ms)**：延迟一般，可正常使用。
   *   **红色 (延迟大于300ms 或超时)**：延迟较高，可能出现卡顿。
*   **智能节点排序与正则表达式过滤**：点击顶部工具栏的齿轮图标：
   *   支持按默认、按名称、按延迟（低延迟自动置顶）或按质量评分自动对节点排序。
   *   支持输入正则表达式（如屏蔽包含“流量”或“官网”字符的节点），直接过滤垃圾节点。

---

## 3. 订阅页 (Subscription)
管理您的订阅源，并负责将其与端口配置、分流模板一键生成最终内核配置文件。

*   **常规端口配置**：支持自定义混合代理端口、外部控制面板端口以及 TProxy 透明代理端口。
*   **控制面板密钥**：配置外部控制器的连接密钥，带眼睛切换显隐。
*   **订阅增删改查**：支持添加多个机场链接，支持配置定时自动更新时间、健康检查（测速）频率，并支持输入过滤前缀（只下载包含该前缀的节点）。
*   **选择分流模板**：在 Base (标准基础版) 和 Full (完整增强版) 中进行选择。
*   **保存并应用**：修改后必须点击底部的「保存并应用」按钮，系统才会生成全新的 `config.yaml` 配置文件并发送热重载命令，让内核无感切换。

---

## 4. 规则页 (Rules)
展示并管理当前内核的匹配规则与分流路由。

*   **实时规则检索**：可在顶部搜索框输入目标域名，检索其命中的分流规则与出站策略。
*   **分流规则临时开关**：每条规则右侧都有一个开关。您可以通过开关随时停用或启用某条特定规则，无需修改底层模板。
*   **规则提供商更新**：支持对 Rule Providers 进行单个更新或一键全部更新（并发上限限制为 3 个并发 Workers）。

---

## 5. 连接页 (Connections)
监控正通过内核代理的活动网络连接。

*   **双列表**：提供「活跃连接」和「已关闭连接」标签页。活跃连接支持以每秒差分法动态计算每条连接的瞬时网速；已关闭连接历史最多保留最近 100 条。
*   **多列排序与搜索**：支持按速度、总流量、连接时间、目的地址进行升序或降序排列，以便快速发现异常跑流量的设备。
*   **连接断开与切换修复**：切换新节点后，如果您想让浏览器连接立即走新节点，可以点击顶部的「断开所有连接」按钮，强制内核断开所有当前连接并重新握手。

---

## 6. 日志页 (Logs)
展示底层的 Mihomo 运行日志。

*   **日志级别过滤**：可按 debug、info、warning、error 过滤展示特定等级日志。
*   **日志控制**：支持搜索过滤，支持随时点击“暂停/恢复滚动”，方便查阅特定日志细节。

---

## 7. 配置页 (Config)
控制底层的内核系统运行。

*   **运行状态控制**：提供一键启动、停止、重启内核的按钮。
*   **内核系统参数**：控制允许局域网连接、启用 IPv6、修改代理模式、修改日志详细度以及绑定特定的物理网卡。
*   **透明网关与例外**：在此页可以直接开启 TProxy 透明代理（必须绑定非 0 监听端口），并点击例外列表弹窗配置绕过代理的局域网主机 IP 或目的端口。
*   **高级清理**：分别提供一键清空 DNS 缓存和一键清空 FakeIP 缓存按钮，用以解决因 DNS 解析污染导致部分网页打不开的问题。
*   **内置 DNS 工具**：提供直观的 DNS 解析测试工具。输入域名和类型（A / AAAA），点击即可查看内核解析出的具体 IP，并在查完点击空白处时自动收起手机虚拟键盘。
